### Decisão Crítica: Sandbox OFF Permanentemente
**Decisão do Carlos:** Manter `agents.defaults.sandbox.mode = "off"` permanentemente.

**Motivo:** Preferência do utilizador por performance/acesso total ao host.

**Implicações:**
- Sessões correm diretamente no host (ia-zeus)
- Acesso total ao sistema de ficheiros e processos
- Modelos locais (qwen3-coder:30b) têm acesso irrestrito a web tools
- Alerta de segurança CRÍTICO permanece ativo no `openclaw status`

**Como reverter se necessário:**
```bash
openclaw config set agents.defaults.sandbox.mode all
openclaw config set tools.deny '["group:web","browser"]'
openclaw gateway restart
```
